## 构造函数（Constructor）

```java
class Human(){
    int age;
    boolean gender;
    Human(){ //default constructor
        age = 0;
        gender = false;
    }
    Human(int age){ //overloading
        gender = true;
        this.age = age;
    }
    Human(boolean gender){
        this(0) //calling constructors from constructors
        this.gender = gender;
    }
}

```
